GBIND - Game Boy Is Not Dead
A GameBoy, GameBoy Color and Super GameBoy Emulator

 

 

Emulator Information: Portuguese Only

 *******************************************
 GBIND - Game Boy Is Not Dead v1.03
 (c) MDashK

 30-07-2008

 Baseado no emulador SMYGB de Ming-yu Shih
 *******************************************

 GBIND é um emulador de GameBoy. Um emulador é um programa que corre num PC e
 simula/imita outro tipo de hardware e software.
 Neste caso, este emulador irá simular um gameboy, criando a possibilidade de jogar jogos de gameboy no PC.

 Este programa só funciona em PCs Windows (NÃO SUPORTA WINDOWS VISTA).

 O GBIND foi criado com base no emulador SMYGB. O que significa que tem as mesmas funcionalidades que o SMYGB.

 GBIND possui suporte para Gameboy, Gameboy Color, Super Gameboy e Super Gameboy Color.
 Tal como o SMYGB, o GBIND funciona em base de DirectSound e DirectDraw.

 ********************************************

 Funcionalidades:
 ----------------

 O GBIND possui suporte para ROMs *.gb. Se possui ROMs de extenção *.gbc terá que seleccionar a
 opção "All Files" na janela de selecção de ROM.
 A causa disto é devido ao emulador não suportar GameBoy Color a uma compatibilidade decente.

 O GBIND tem:
 - suporte para jogar em diferentes tamanhos de janela, inclusive ecrã inteiro.
 - Gráficos e Som idêntico ao Gameboy.
 - Suporte para Joystick
 - Através da opção "gravar estado/carregar estado", pode gravar o jogo, em tempo real, de modo a se tiver
 gravado e por exemplo, perder uma vida, carregar o estado anterior e voltar ao ponto antes de perder essa vida.
 Esta opção grava o estado não só na RAM, mas também num ficheiro do tipo *.gbn que será gravado
 na mesma pasta onde se encontra a ROM. Assim, se necessitar de fechar o emulador, desligar o PC e
 querer continuar o jogo mais tarde, poderá fazê-lo. Tecnicamente falando, esta opção gravará a RAM actual do
 gameboy para um ficheiro que poderá ser utilizado mais tarde.
 - suporte para definir prioridade de execução
 (COLOCAR EM MODO DE TEMPO REAL PODERÁ CAUSAR PROBLEMAS COM OUTRAS APLICAÇÕES)
 - Opção para saltar frames (caso o seu PC dê o emulador lento, ao saltar frames,
 a velocidade de jogabilidade poderá melhorar)


 Requerimentos do sistema:
 -------------------------

 O GBIND é um emulador rápido, simples e eficiente, significando que não necessita de
 muita potência para correr a 100%.

 Necessita de no mínimo:
 - Um PC Intel Pentium 166 (Pentium II recomendado)
 - A sua placa gráfica tem de possuir suporte para DirectDraw (tem de correr em modo 16 bit)
 - A sua placa de som tem de possuir suporte para DirectSound
 (é obrigatório possuir placa de som para correr o emulador).


 Controles:
 ----------

 Gameboy -> PC
 .............
 Start 		-> 	Tecla Enter
 Select 	-> 	Tecla Shift
 A 		-> 	Tecla X
 B 		-> 	Tecla Z
 Direcção 	-> 	Teclas de Setas

 Para alternar entre janela e ecrã inteiro, carregar em Alt+Enter ao mesmo tempo.



 Problemas conhecidos:
 ---------------------

 - Quando se grava um estado, e se faz carregar estado, o som do jogo estará incompleto durante um breve momento.
 - Ao mudar de ecrã inteiro para janela, poderá ocorrer a situação de os objectos do ambiente de
 trabalho aparecerem fora do lugar onde anteriormente estavam.
 - GBIND NÃO SUPORTA FORMATO ZIP OU RAR. OS ROMS TERÃO DE ESTAR DESCOMPACTADOS.
 - GBIND NÃO SUPORTA WINDOWS VISTA.
