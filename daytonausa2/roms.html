In order to keep this site free of legal issues, we can't, unfortunetally, share the ROM files of the game. But, they are easily found using Google... There are various websites that share the ROM files. Just use the filenames we provide.

As of 24/06/2015, the latest version of the game is "Supermodel SVN r274". Just Google it, and you'll easily find it. Also, I advice using "Supermodel-UI v0.9 build 151" as a front-end for the emulator.

Since April 1, 2011 there's been a new emulator of Model 3 in the internet that puts MAME to a corner.
I'm talking about the new in-progress SuperModel emulator, a Sega Model 3 Arcade Emulator, better yet, THE Sega Model 3 Arcade Emulator.

I must send a VERY BIG SPECIAL THANKS to Scott (Gamertag: AM2Model3).
He was the one that e-mailed me right here from the site informing me about the release of this emulator. =)
Big special thanks to him and to all the Daytona USA 2 fans that keep visiting the website.

Some advice from Scott when running the emulator:
When you run the emulator, (if you don't use a GUI front-end) I would suggest using this command line to get the best performance.
I can get 60FPS in HD resolutions:

supermodel [rom_file] -res=640,480 -fullscreen -no-throttle -ppc-frequency=50

This should give you the best performance on any system. You can change the rom_file name and the resolution. 640x480, 1024x768, 1280x1024, 1280x720, 1920x1080...

So here you have it. =) BTW: Make shure you have a nicely-powered machine to run this emulator, because it really pushes your graphics and processor up. You should be fine using, for example, my setup: Intel i5 CPU with a GeForce GT GPU and 4GB RAM.

And finally, but not least: A VERY BIG SPECIAL THANKS to Bart Trzynadlowski and all the Team involved in the emulator, for sharing with us this awesome emulator.
